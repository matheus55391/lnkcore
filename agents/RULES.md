# Agent Rules Source

Este diretorio concentra as regras de comportamento para agentes no projeto.

## Prioridades

1. Seguir TDD sempre (Red -> Green -> Refactor).
2. Sugerir testes antes de implementar features.
3. Nao colocar regra de negocio em componentes de UI.
4. Validar ownership e validacao server-side em qualquer operacao de dashboard.
5. Manter arquivos pequenos e modulares por dominio.
